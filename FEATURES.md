# My Money — 功能清單與實作狀態

> 最後更新：2026-04-01

## 總覽

個人財務管理 app（Flutter Web + iOS + Android），追蹤銀行帳戶、信用卡、固定支出攤提、儲蓄目標、現金流預測。

後端：Cloudflare Workers + D1 Database
